1. Addition, subtraction - все традиційно, "в стовпчик".
2. Multiplication:

     y ^          |               |
       |    TC3   |      TC3      | TC3
  7680 |__________|_______________|______
       |          |               |
       |          |               |
       | Стовпчик |   Karatsuba   | TC3
       |          |               |
  2560 |__________|_______________|______
       |          |               |
       | Стовпчик |   Стовпчик    | TC3
       |          |               |
       |----------|---------------|------>
                 2560            7680    x

3. Division:
    - Якщо довжина дільника менше 80 * 32 = 2560 біт або
      різниця довжин між діленим та дільником менше 40 * 32 = 1280 біт,
      то використовується алгоритм Кнута
    - Інакше - алгоритм Бурнікеля-Циглера.

4. Mod Inverse - різні алгоритми, в залежності від парності/непарності числа та модуля.
                 Для комбінації результатів використовується КТЛ.

5. Mod Pow - окремий алгоритм для непарного модуля.
             Якщо модуль парний, то виділяється 2^t та непарна частина,
             окремо для обох частин рахуються результати і комбінуються
             через КТЛ.
